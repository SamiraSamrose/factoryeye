Welcome to FactoryEye Demo

This interactive demo allows you to test all components of the FactoryEye system. Explore the multi-agent architecture, simulate sensor data, trigger anomaly detection, and see how the system responds to various scenarios.

System Components

1
Sensor Sentinel
Real-time anomaly detection agent running on GPU. Processes sensor streams using Gemma model and flags anomalies within seconds.
2
Predictor AI
Batch predictive maintenance agent. Analyzes historical patterns to forecast equipment failures and calculate Remaining Useful Life (RUL).
3
Efficiency Analyst
Generates natural language insights using Gemini and creates visualizations with Imagen. Provides actionable recommendations.

Data Flow Visualization

Understanding how data flows through the system is crucial. Below is the complete pipeline:

  1. IoT Sensors generate telemetry data (temperature, vibration, pressure, RPM)
  2. MQTT Broker receives sensor messages and forwards to Cloud Pub/Sub
  3. Pub/Sub provides reliable message queuing and delivery
  4. Sensor Sentinel pulls messages and runs anomaly detection
  5. BigQuery stores all readings, anomalies, and predictions
  6. Predictor AI runs hourly batch jobs for RUL predictions
  7. Efficiency Analyst generates insights and visualizations on demand
  8. Dashboard displays real-time metrics and historical trends
